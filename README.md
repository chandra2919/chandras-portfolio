## Tech Stack

- **HTML5** — semantic structure
- **CSS3** — custom properties, grid, flexbox, keyframe animations
- **JavaScript (Vanilla)** — neural canvas, scroll animations, page transitions, stat counters

---

## Features

- ⚡ Animated neural network canvas (home page)
- 🎨 Dark theme with electric blue accent (`#0A84FF`)
- 📱 Fully responsive — mobile-first breakpoints at 960px, 768px, 480px
- 🔄 Smooth page fade-in / fade-out transitions
- 🃏 Visual project cards with gradient panels, icon badges, and stat chips
- 📊 Scroll-triggered stat counters and fade-up animations
- 🔵 Animated CGPA ring on Education page
- 🖼️ Ambient orb backgrounds and dot-grid overlays on inner pages
